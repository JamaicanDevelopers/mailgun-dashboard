# mailgun-dashboard

PoC dashboard to play with [Mailgun API](https://github.com/mailgun/mailgun-go) to view message content and resend messages

## Usage

```bash
go get -u -v
go build
MAILGUN_DOMAIN=<domain> MAILGUN_APIKEY=<api key> ./mailgun-dashboard
```
